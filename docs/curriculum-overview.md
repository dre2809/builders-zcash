# 📚 Curriculum Overview

This program is designed to onboard developers into the Zcash ecosystem through a structured and practical approach.

Zcash leverages zero-knowledge proofs (zk-SNARKs) to enable private and verifiable transactions.

---

## 🎯 Key Focus Areas

- Privacy-first design thinking
- Practical development skills
- Open-source contribution pathways
- Community building

---

## 🧱 Learning Approach

The curriculum is divided into 6 phases:

1. Foundations & Architecture
2. SDKs & APIs
3. Application Development
4. Hands-on Workshops
5. Testing & Security
6. Open Source Contribution
